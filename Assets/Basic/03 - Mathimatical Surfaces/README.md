# Mathimatical Surfaces

[The source article](https://catlikecoding.com/unity/tutorials/basics/mathematical-surfaces/)

The goal of this is to learn about:

1. Create a function library
2. Use a delegate and an enumeration type
3. Display 2D functions with a grid
4. Define surfaces in 3D space

### Notes

- The `delegate` keyword in C# is how one declears a function pointer type. For example `delegate [return_type] [function_name](...);`
